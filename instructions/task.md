Task: Build a Mini Library Management System (LMS)  

Stack: 
- .NET 8.0 (backend API services)
- React (frontend, UX)
- Docker
- AWS


Minimum Features of LMS: 

- Book Management: Add, edit, and delete books (title, author + whatever metadata you see fit)
- Check-in/Check-out: Mark books as checked in (borrowed) or checked out (returned) 
- Search: Find books by title, author, or other fields
- Authentication system with SSO, preferably with different user roles and permissions
- Brief README describing how to run it


Anticipate deployment to AWS as a simple web application, include infrastructure-as-code artifacts for spinning up AWS environment and relevant resources.


Once completed basic LMS, read through bonus.md and follow instructions.
